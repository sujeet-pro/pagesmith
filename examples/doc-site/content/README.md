# Welcome to Pagesmith

Pagesmith is a file-based CMS and static site generator. It provides schema-validated content collections, a powerful markdown pipeline, and zero-config static site generation.

## Getting Started

Check out the [Guide](/guide/getting-started) to learn the basics, explore the [API](/api/content-layer) for detailed reference, or browse the [Reference](/reference/schemas) for configuration options.
