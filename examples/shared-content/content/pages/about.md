# About This Example

This site demonstrates how to use `@pagesmith/core` with a JavaScript framework to build a content-driven website.

## What it shows

- **Content collections** — typed markdown content with Zod schema validation
- **Vite integration** — virtual modules for importing collections in your components
- **SSR pre-rendering** — static HTML output for every route
- **Pagefind search** — full-text search indexed at build time
- **Responsive layout** — 3-column grid with sidebar navigation and table of contents

## Stack

- `@pagesmith/core` for the content layer
- Vite for bundling
- Pagefind for search
- CSS custom properties for theming (light/dark via `light-dark()`)
